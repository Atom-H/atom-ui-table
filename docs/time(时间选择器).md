# time(时间选择器)   


![](https://github.com/MaiYuan/Admin5/blob/master/docs/images/time.png?raw=true)


### 数据说明
|键名 |说明 |
| ------------ | ------------ |
|title| 标题|
|is| 必须"time"  |
|value| 默认值  |
|name   | ajax字段名, 服务端接收参数名  |
|column   | 对应bootstrap的栅格布局  |

### 数据示例
``` json
{
    "title": "时间选择器",
    "is": "time",
    "value": "08:20",
    "name": "timePickValue",
    "column": [
        12,
        12,
        2,
        2
    ]
}
```
